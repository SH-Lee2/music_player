
# 🎧 Music Player

👉 url : https://pre-music-player.netlify.app/

## 🚀 intro: 서비스 소개
장르별 노래, 사용자 나라의 노래 차트, 가수와 노래의 인기순 노래를 들을수 있는 사이트 입니다.
